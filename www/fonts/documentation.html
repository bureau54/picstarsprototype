WordPress Installation

  1. Log-in to your WordPress site
  2. Navigate to the Plugins Menu > Click Add New > Click the Upload link
  3. Click Choose File > Locate the iOS7 Style Font Icon .zip file > Click Install Now
  4. Click Activate Plugin
  5. Create / Edit a Page or Post and select your icons via the "Font Icon" Select Menu within the Editor.

Add iOS7 Style Font Icons to a HTML site

1. Copy the following folders from the plugin folder:

  • css
  • fonts
  • js

2. Upload them to your site. e.g.

  • http://your-site-name.com/css
  • http://your-site-name.com/fonts
  • http://your-site-name.com/js

3. Add the following HTML into the <head> section of your HTML page:

<link rel="stylesheet" href="css/ios7-style-font-icons.min.css" />
<!--[if lte IE 7]><script src="js/ios7-style-font-icons-ie7.js"></script><![endif]-->

Add an Icon to WordPress Pages / Posts

  1. Create / Edit a Page or Post
  2. Within the Page / Post Editor, locate the "Font Icons" select menu.

  3. Select the Icon you wish to place into your Page / Post.

 

HTML Code + WordPress Shortcode

Place icons almost anywhere with the <i> or <span> HTML tags. e.g.

HTML: 

<i class="icon-book"></i>

Shortcode: 

[icon name="icon-book" /]

Unicode Example

Visit the Font Icons page and hover over the icon you wish to grab the unicode from, copy the unicode. e.g. E17C

HTML: 

<span class="fastforward"></span>

CSS: 

span.fastforward {
	font-family:"ios7-style-font-icons";
}
span.fastforward:before {
	content: "\E17C";
}

Set Icon Color + Size

  1. For WordPress, switch to Text view, for HTML, simply edit the HTML.

     

  2. SIZE - To change the size, add an extra class after the icon name e.g.

    HTML:
    <i class="icon-book icon-2x"></i>

    Shortcode:
    [icon name="icon-book icon-2x" /]


    Choose from the following sizes icon-large , icon-2x , icon-3x , icon-4x


  3. COLOR - To add a color to your icon using HTML, simply add a bit of inline CSS styling. For the WordPress Shortcode, add the iconcolor attribute.

    Set the hex color ( #60b5f6 ) to the color you require. e.g.

    HTML
    <i class="icon-book" style="color:#60b5f6;"></i>

    Shortcode:
    [icon name="icon-book" iconcolor="#60B5F6" /]

Stacked Icons

Stacked icons is a method for easily stacking multiple icons on top of each other using the icon-stack class on the parent and icon-stack-base for the bottom icon.

<span class="icon-stack">
	<i class="icon-check-empty icon-stack-base"></i>
	<i class="icon-beaker"></i>
</span>

	<i class="icon-circle-blank icon-stack-base"></i>
